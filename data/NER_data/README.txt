Synthetic domain-aligned NER dataset for natural-disaster alert generation.
Total sentences: 300
Splits: train=240, dev=30, test=30
Format: token-level CSV with columns sentence_id, token_id, token, tag
Labels: DISASTER, LOCATION, CASUALTY, INFRASTRUCTURE, AID_RESOURCE, ORG
BIO tagging scheme.
Language: English.
Notes: This dataset was generated to be aligned with the CrisisMMD domain, not extracted directly from CrisisMMD.
